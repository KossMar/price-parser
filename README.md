# price-parser
An application created to stay up to date on cryptocurrency prices

## Getting Started

### Installation
```
go get -d github.com/kossmar/price-parser
```

## Flags

| Flag             | Function                                                                     |
|------------------|------------------------------------------------------------------------------|
| -v               | vebose output                                                                |
| -j               | json output                                                                  |
| -T               | display time between outputs                                                 |
| --api            | specify cryptocurrency service                                               |
| --coin           | specify cryptocurrency                                                       |
