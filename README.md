# price-parser

### Usage

##### Download
```
go get -d github.com/kossmar/price-parser
```

### Flags

| Flag             | Function                                                                     |
|------------------|------------------------------------------------------------------------------|
| -v               | vebose output                                                                |
| -j               | json output                                                                  |
| -T               | display time between outputs                                                 |
| --api            | specify cryptocurrency service                                               |
| --coin           | specify cryptocurrency                                                       |
